`synchronized` 关键字在方法上使用时，在方法修饰符上增加了一个标志位 `flags: (0x0021) ACC_PUBLIC, ACC_SYNCHRONIZED`。而用在代码块时，则是生成了 `monitorenter` 和 `monitorexit` 指令。
